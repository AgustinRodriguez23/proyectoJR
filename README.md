# 🎵 Proyecto Web – Artista Musical

Este proyecto es una página web simple dedicada a un artista musical.
Permite visualizar información del artista, acceder a sus canciones y leer las letras de cada una.

## 📌 Descripción

La página está diseñada para mostrar contenido de un artista de forma clara y accesible.
Al ingresar al sitio se puede ver:

* Imagen o logo del artista
* Breve descripción
* Lista de canciones
* Acceso a la letra de cada canción
* Links a redes sociales y plataformas

El objetivo del proyecto es practicar **HTML, CSS y diseño responsive**, creando una experiencia visual similar a plataformas musicales.

## 🚀 Funcionalidades

* 🎤 Imagen principal del artista
* 📝 Descripción corta del artista
* 🎶 Lista de canciones
* 📄 Página individual con la letra de cada canción
* 📱 Diseño responsive para móviles
* 🎧 Estilo visual inspirado en Spotify
* 📷 Íconos de redes sociales
* 🔗 Link al Instagram del artista
* ▶️ Link al canal de YouTube

## 🛠️ Tecnologías utilizadas

* **HTML5**
* **CSS3**
* **Font Awesome** (para íconos)
* **Responsive Design**

El sitio está optimizado para:

* 📱 celulares
* 💻 computadoras
* 📟 tablets

Utilizando **media queries en CSS**.

Este proyecto fue creado con fines de aprendizaje para:

* Practicar **maquetado web**

* Mejorar habilidades en **HTML y CSS**
